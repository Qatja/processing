Qatja

A library by Andreas Göransson for the Processing programming environment.
Last update, 03/02/2014.

Allows your processing sketch to connect to MQTT servers.

Download

Download Qatja version 0.1.1 (1) in .zip format.

Installation

Unzip and put the extracted Qatja folder into the libraries folder of your Processing sketches. Reference and examples are included in the Qatja folder.

Keywords. mqtt
